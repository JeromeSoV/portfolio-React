## Description du projet

Recréation de mon portfolio web, anciennement en Astro, en React. Apprentissage de React en autodidacte, 
avec comme support le tutoriel en ligne de la W3 School et l'utilisation de Claude dans le but d'appliquer les notions
apprises avec des exemples et exercices complémentaires au tutoriel.
L'apprentissage du framework m'a pris une semaine, et l'implémentation des features les plus importantes ont été fait dans la semaine suivante.
L'implémentation de futures fonctionnalités et de changements mineurs se feront progressivement, en parallèle de mes autres projets.
Le portfolio n'est pas encore dans sa version finale, des améliorations sont encore prévues, comme la mise en place d'un carousel,
l'implémentation d'un outil de recherche dans la page projet, et la gestion d'un formulaire de publication de projet 
(stockés en session, pour ne pas altérer le but du portfolio), ainsi que d'autres améliorations graphiques et de responsiveness globale.
Une logique backend d'archivage des projets a été mise en place, avec des scripts TS permettant la transformation de fichiers Markdown
en fichiers JSON utilisables en frontend, que ce soit pour afficher la liste des projets, et le détail d'un projet.
La logique backend est destinée à être évacuée dans un autre projet d'apprentissage du NoSQL (MongoDB) et du framework Express.js, afin de faire une API de gestion d'archive.


## Compétences

* React
  * Logique de réalisation d'application, en particulier de SPA
  * Composants
  * Gestion des états et des props
  * appels AJAX
  * Routing
* TypeScript
* Node.js
  * Gestion de projet
  * Utilisation de librairies : Gray Matter, Remark, fs, path

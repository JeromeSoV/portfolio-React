## Description du projet

Création d'un portfolio personnel en ligne.
Le site a été développé en utilisant le framework Astro. Utilisation de mes connaissances passées en HTML et CSS.
Premières utilisations de JavaScript pour le développement web.
Achat d'un domaine personnel et d'un VPS pour héberger le site, utilisation de Nginx.
Utilisation de LLM tel que ChatGPT pour des retours et avoir des suggestions d'approfondissements : 
L'utilisation du framework Astro m'a été conseillé afin de répondre à un besoin de factorisation du code que je ressentais
 qui fut résolu par l'utilisation de composants.
Cette première version avait plusieurs fonctionnalités de prévues, tel qu'une gestion automatisée des projets, qui fut géré
par la suite dans la deuxième version qui utilise le framework React.

## Description du Projet

Ce projet consistait en la création d'une API REST en Java.
Le sujet reposait sur la gestion d'un système de collecte de déchets avec 
une notion de score utilisateur selon les déchets déposés.
Le sujet évoluait chaque semaine, avec à chaque fois une extension de notre base de données,
des ajouts d'endpoints à apporter, de nouveaux cas d'erreur à traiter, avec un contrôle des
résultats et des erreurs, assurés par l'utilisation du logiciel Bruno.

## Compétences

- REST
    - Développement d'API
    - Gestion des différentes méthodes HTTP
    - Gestion des codes de retour et erreurs
    - Gestion d'authentification via Basic Auth
- SQL
    - Création de requêtes avancées: window functions, gestion des nulls, etc.
- Bruno
    - Validation de résultats et erreurs
    - Utilisation de scénarios de test
